Jogo da Velha em C

Este projeto é uma implementação simples do jogo da velha em linguagem C. O jogo utiliza uma matriz 3x3 para representar o tabuleiro e permite que dois jogadores, X e O, joguem alternadamente, com a possibilidade de inserir seus próprios nomes.

Funcionalidades

Tabuleiro 3x3 : Utilize uma matriz 3x3 para representar o tabuleiro do jogo.
Alternância de Jogadores : Permite que os jogadores 'X' e 'O' alternem seus turnos, com opção de inserir seus nomes.
Verificação de Vencedor : Identifica se um jogador ganhou o jogo, anunciando o nome do vencedor.
Verificação de Empate : Determina se o jogo terminou em um empate.
Validação de Entrada : Garanta que todas as jogadas sejam válidas e dentro do tabuleiro.
Interface Personalizada : Interface de usuário no terminal para entrada de nomes e saída de dados.
Jogo Continuado: Permite ao usuário escolher se quer jogar novamente ou não.

Como jogar

1- Execute o programa.
2- Os jogadores serão solicitados a inserir seus nomes e se quer começar com X ou 0.
3- Os jogadores se alternam, inserindo a linha e a coluna onde desejam colocar seu símbolo (X ou O).
4- Caso a jogador escolha uma posição já ocupada ou inserir números fora do que foi pedido, o jogo pede que ele refaça a jogada! 
5- O jogo prossegue até que haja um jogador vitorioso que consiga alinhar três símbolos em linha, coluna ou diagonal.
6- Caso o tabuleiro esteja completamente preenchido haverá um empate.
7- O jogo pergunta se quer jogar novamente
8- Caso queira jogar novamente, o jogo apaga o jogo anterior para melhor visualização.

O nome do vencedor é mostrado no final do jogo, caso haja vencedor.
